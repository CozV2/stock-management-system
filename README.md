# 📦 Stock Management System

A desktop inventory management application built with **C# (Windows Forms)** and **MySQL**.

This project allows users to manage products, track stock movements, and export inventory data to Excel.

---

## 🚀 Features

- Add new products
- Edit existing products
- Delete products
- Automatic stock movement tracking
- Stock history (IN / OUT movements)
- Product search functionality
- Export product list to Excel
- Clean and simple desktop interface

---

## 🛠 Technologies Used

- C#
- .NET Framework
- Windows Forms
- MySQL
- Visual Studio
